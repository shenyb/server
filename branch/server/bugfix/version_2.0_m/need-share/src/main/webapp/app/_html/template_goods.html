现学烘焙太难做

by Zoey

看到精致可口的让人垂涎欲滴的点心，就心痒难耐的想叫上闺蜜一起来个下午茶，边吃边聊。本着会吃就会做的吃货精神，也曾深入厨房，苦学了一段时间，奈何学艺不精，还是买着吃更能满足自己挑剔的嘴巴。

那段时间，最常看的是一个法国老奶奶莫妮卡的视频，她和他的孙子大卫一起制作了 100 个美味点心的制作视频，包括了传统提拉米苏，马卡龙，彩虹堂曲奇，肉桂苹果挞等等超级美味的点心。
